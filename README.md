# Smart Grid Cutter - Cắt Ảnh Thành Lưới

Ứng dụng web cắt ảnh thành nhiều phần nhỏ theo lưới tùy chỉnh, hỗ trợ xem trước trực quan và tải nhanh.

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt
2. **Tải ảnh**: Click vùng "📷 Tải ảnh" hoặc kéo thả ảnh
3. **Điều chỉnh lưới**: Dùng thanh trượt Hàng và Cột (1-10)
4. **Tùy chỉnh lưới**:
   - 🎨 Chọn màu lưới (trắng/đen)
   - 📏 Điều chỉnh độ dày đường lưới
5. **Cắt ảnh**: Nhấn "✂️ Cắt"
6. **Tải kết quả**: Lưu từng ảnh, tải tất cả, hoặc tải ZIP

## Tính năng

- **Tải ảnh**: Click hoặc kéo thả, hỗ trợ JPG, PNG
- **Preview trực quan**: Xem lưới cắt trước khi cắt
- **Thanh trượt thập phân**: Chia lưới chính xác (1-10, bước 0.01)
- **Gợi ý thông minh**: Hệ thống đề xuất layout đẹp nhất
- **Cảnh báo cắt hình**: Cảnh báo khi lưới đi qua vùng có hình ảnh
- **Tùy chỉnh lưới**: Màu sắc và độ dày đường lưới
- **Tải linh hoạt**: Từng ảnh, tất cả, hoặc ZIP
- **Giao diện thân thiện**: Responsive, hướng dẫn sử dụng tích hợp

## Yêu cầu

- Trình duyệt hiện đại (Chrome, Firefox, Edge, Safari)
- Không cần cài đặt, chạy trực tiếp từ file HTML
